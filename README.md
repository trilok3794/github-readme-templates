# GitHub README Templates

A collection of simple GitHub README templates for open source projects, personal portfolios, documentation sites, SaaS products, and developer tools.

Good README files help users understand your project, installation process, features, and usage instructions quickly.

---

## Basic Project README

```md
# Project Name

Short description of your project.

## Features

- Feature 1
- Feature 2
- Feature 3

## Installation

npm install

## Usage

npm start

## License

MIT
```

Setup instructions.

---

## README Best Practices

* Keep the introduction short.
* Add installation instructions.
* Include usage examples.
* Add screenshots when useful.
* Keep formatting consistent.
